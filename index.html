Selected Projects

CertiKOS

    To fully verify CertiKOS (a realistic kernel hypor-visor), we introduce 37 abstraction layers. Each abstraction layer specifies the contextual behaviors of primitives introduced at underlay. By linking all the layers, we show that there is a contextual refinement between any user program P linked with abstract system call primitives (which are atomic), and P linked with the kernel implementation.

Verification of uC/OS-II

    Proposed a new abstract interrupt machine, named uAIM, to verify the schedule function of uC/OS-II , which is a widely used real-time kernel. Our new framework successfully verified inline assembly code in C, nested critical section, as well as the preemptive scheduling.

    I presented this work as my undergraduate honors thesis.
