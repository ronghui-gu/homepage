  • “Eric is younger than Ronghui”.

1.2 Introduction of First Order Logic

First Order Logic := Propositional Logic + :

  • Terms: syntax only, e.g., “Eric”, “Ronghui”, etc.
  • Predicates: property of terms:
    • (Eric): Eric is a student
    • (Eric, Ronghui): Eric is younger than Ronghui
  • How to talk about every student？
    • FOL uses variables and universal quantification
    • : “all terms are students”
    • : “forl all terms who are students, …”
  • How to talk about the existence of at least one Instructor
    • FOL uses variables and existential quantification
    • : “there exists a term which is an Instructor and Eric is younger than this instructor”.
